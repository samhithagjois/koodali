# koodali 

Hello, welcome to the GitHub repository of Koodali, Namma Kannada Shaale's administration tool!
This repository was created and will be maintained by Samhitha. 
This project is a final-year Computer Science full-stack development project. The backend is written with Java and the frontend uses ANgular

Right now it is still in development phase. There are little to no CSS involved, and only rudimentary HTML structures used. 

To run the server, clone the repository and run the KoodaliAppApplication. 
To run the frontend server, run the Angular CLI server with ng serve.
